Hotel Revenue Analysis

In this project i cleaned and analyzed a hotel data. For this project i aim to answer some business question
1. Is our hotel revenue growing by year? (Keeping in mind that we have two types of hotel)
2. Should we increase our parking lot size.
3. What customer type generate more revenue?
4 . What trends can we see in the data?
There are five tables in the data set. Firstly I extracted the data to SQL where I cleaned the data, applied CTE, UNION, LEFT JOIN statements, Numeric expressions to make the data more usable. Then I connected the SQL queries to power BI for visualization.
